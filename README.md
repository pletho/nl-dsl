# Strinum으로 명칭변경

nl-dsl 형태의 일환인 트리구조선언형언어인 StrinStrum을 중복된 명칭과 길이를 줄이고자 '스트리넘(Strinum)'으로 재명명하여
레파지토리도 변경한다

https://github.com/pletho/strinum 
으로 현시점부로 이동한다.
